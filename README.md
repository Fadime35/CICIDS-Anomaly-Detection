# CICIDS-Anomaly-Detection

This project detects network anomalies using the CICIDS2017 dataset.

## Dataset

CICIDS2017 network traffic dataset is used in this project.

## Technologies

- Python
- Pandas
- NumPy
- Scikit-learn

## Features

- Network traffic preprocessing
- Benign traffic extraction
- Data normalization
- Preparation for deep learning models
